# 一百分 AI 程式課程 · 給會計師的 AI 程式入門

4 小時完整課程，從零教會計師用 AI 做出客戶看得懂的互動網頁。

## 📁 內容

- **`00_課程總覽.txt`** — 完整教案（含口白、prompt、圖片清單）
- **`01_課前作業.html`** — 開課前 1 週發給學員自學
- **`02_課程簡報.html`** — 上課用簡報（鍵盤 ← → 翻頁）
- **`assets/course-art/`** — 課程插畫素材
- **`示範作品/`** — 5 個現場示範作品
  - `憑證偵探/` — 找錯誤遊戲
  - `公司vs行號/` + `公司vs行號_設計版/` — 拖拉配對
  - `營業稅試算/` — 滑桿即時試算
  - `圖文動態三段對比/` — 章節 7-4 配套示範
  - `轉蛋機/` + `轉蛋機_完成版/` — 任務獎勵轉蛋

## 🎓 課程章節

```
章節 1   LLM 是大腦，IDE 是手腳
章節 2   IDE vs 終端機
章節 3   Antigravity + Claude 串接
章節 4   第一次動手做小遊戲
章節 5   怎麼跟 AI 溝通
章節 6   互動示範作品集
章節 7   遊戲化原理（含轉蛋機示範）
章節 8   從零做網頁 + GitHub + 進階知識
章節 9   跟 AI 長期協作的 3 件事
章節 10  結業 · 結語
章節 11  AI 小知識補充
```

## 🚀 怎麼用

**備課**：先讀 `00_課程總覽.txt`，把標 `[📷 需補圖]` 的地方準備好截圖。

**開課前 1 週**：把 `01_課前作業.html` 傳給學員自學。

**上課**：開 `02_課程簡報.html` 全螢幕投影，鍵盤 ← → 翻頁。

**示範時**：講到對應章節時，新分頁打開 `示範作品/` 對應的 `index.html`。
