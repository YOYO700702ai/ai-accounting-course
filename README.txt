🚀 怎麼用

備課時：
  1. 先讀 00_課程總覽.txt 從頭到尾走一遍
  2. 標註 [📷 需補圖] 的地方準備截圖
  3. 標註 [🔗 示範作品] 的地方確認對應檔案可以開啟
  4. 自己跑一遍每個 prompt，確認 AI 產出符合預期

開課前 1 週：
  把 01_課前作業.html 傳給學員自學

上課當天：
  - 開 02_課程簡報.html 全螢幕投影
  - 鍵盤 ← → 翻頁
  - 講到示範作品時，新開分頁打開對應的 index.html
  - 講到 prompt 時，用簡報上的 prompt 區塊直接複製給學員看

學員結業時：
  把整個「AI會計課程總整理」資料夾打包送給學員當課後參考
