ANTIGRAVITY IDE · 新手安裝指南

Antigravity 需要安裝哪些模組？

先把介面變成看得懂的繁體中文，再選擇一位 AI 程式助理。這份圖文說明依照你的安裝畫面整理，照著做就能完成基本設定。

先看結論開啟延伸模組繁中語言包AI 助理二選一完成檢查

先看結論：建議安裝 2 個

Antigravity 的「延伸模組」就是替 IDE 加上新功能的小工具。剛開始使用時，安裝繁體中文語言包，再從 Codex 與 Claude Code 裡選一個，即可開始使用。

重要：Codex 和 Claude Code 都是 AI 程式助理，功能有重疊。一般使用者不必兩個都裝；選你有帳號、訂閱或老師指定的那一個即可。
建議必裝

繁體中文語言包

把選單、按鈕和提示轉為繁體中文，操作更容易。

二選一

Codex — OpenAI

在 IDE 裡使用 Codex 協助讀程式、修改檔案與執行工作。

二選一

Claude Code — Anthropic

在 IDE 裡使用 Claude Code 協助理解專案與撰寫程式。

第 1 步：找到「延伸模組」

1

點左側的方塊圖示

滑鼠移到圖示時會顯示「延伸模組」。也可按 Ctrl + Shift + X 開啟。

2

在上方搜尋框輸入名稱

例如輸入 Chinese (Traditional) Language Pack、Codex 或 Claude。

3

點選正確項目後安裝

確認作者、名稱和圖示，再按「安裝」。出現「正在安裝」時請稍等。

你的畫面顯示 Antigravity 預設使用 Open VSX 市集。這是正常的擴充功能來源；只要從搜尋結果中確認名稱與發行者即可。

第 2 步：先裝「繁體中文語言包」

建議必裝

Chinese (Traditional) Language Pack for Visual Studio Code

你的截圖顯示發行者是 MS-CEINTL，描述為「Language pack extension for Chinese (Traditional)」。這個模組只負責把介面中文化，不會替你寫程式，也不需要額外登入 AI 帳號。

  • 在搜尋框輸入 Chinese (Traditional) Language Pack。
  • 點選名稱相同、發行者為 MS-CEINTL 的結果。
  • 按「安裝」。若系統提示重新啟動或重新載入視窗，依提示操作即可生效。
  • 看到按鈕變成「解除安裝」，代表已安裝完成。

第 3 步：AI 程式助理二選一

兩個模組都能在開發過程中提供 AI 協助，但它們屬於不同公司與服務。請按你的使用需求選擇，而不是因為看到搜尋結果就全部安裝。

選項 A

Codex — OpenAI's coding agent

截圖中發行者是 openai。適合已使用 OpenAI／ChatGPT 生態系、並希望在 IDE 裡與 Codex 合作完成程式工作的使用者。

  • 搜尋：Codex
  • 確認名稱：Codex — OpenAI's coding agent
  • 確認發行者：openai
  • 安裝後依畫面登入或授權；是否需要可用帳號或方案，以安裝後顯示的流程為準。
選項 B

Claude Code for VS Code

截圖中發行者是 Anthropic。適合已使用 Claude 服務，並希望把 Claude Code 直接整合到 IDE 的使用者。

  • 搜尋：Claude
  • 確認名稱：Claude Code for VS Code
  • 確認發行者：Anthropic
  • 安裝後依畫面登入或授權；是否需要可用帳號或方案，以安裝後顯示的流程為準。

安裝完成後，這樣檢查

  • 搜尋到的項目按鈕已從「安裝」改為「解除安裝」或顯示已安裝狀態。
  • 繁中語言包安裝後，若介面仍是英文，先依提示重新載入／重啟 Antigravity。
  • 只保留一個 AI 助理（Codex 或 Claude Code）作為起點；日後真的需要再加裝另一個。
  • AI 模組第一次開啟時，依官方畫面登入。不要把密碼、API 金鑰或敏感資料貼進一般對話。
  • 安裝失敗時，先確認網路，按延伸模組頁面的重新整理按鈕，再重新嘗試。
