# 《燭之武退秦師・一夜退兵》互動歷史劇場 — 完整施工企畫書 v3

> **本文件用途**:可直接交付給工程師/AI 從零開始建構整個遊戲。已整合多輪修訂,所有規格皆為最終版。
>
> **改編自**:《左傳・僖公三十年》
> **目標族群**:國中、高中國文必讀篇章互動教材
> **核心玩法**:對白選擇 + 月夜潛行 +「論點卡牌・五段對話說服戰」(本作獨創)
> **教育目標**:理解外交說服術的邏輯結構、識人之明、政治家風度、成語「東道主」典故。

---

## 一、總體目標

玩家分別扮演 **燭之武**(主線)與 **晉文公**(支線),依序經歷 12 場戲:

1. 大軍壓境(秦晉聯軍圍鄭)
2. 朝堂議事(佚之狐進言)
3. 茅屋求賢(鄭伯三請燭之武)
4. 往昔閃回(燭之武回憶被埋沒)
5. 夜縋而出(月夜下城)
6. 月下潛行(🎮 反應小遊戲)
7. 秦營外(求見秦穆公)
8. **論點卡對峙**(🎮 核心玩法:五段對話說服戰)
9. 秦伯退兵
10. 子犯請擊(切換晉文公視角)
11. 晉文公的選擇
12. 三軍齊退
+ 尾聲一:東道主成語誕生
+ 尾聲二:燭之武歸隱
+ 隱藏:殽之戰彩蛋(通關後解鎖)

## 二、技術規格

- 原生 **HTML / CSS / JavaScript**,**不使用任何框架**
- **不使用** `localStorage` / `sessionStorage`,所有狀態用 JS 變數
- **16:9 橫式**,**固定設計尺寸 1100 × 618.75 px**
- 手機/平板/桌機:以 CSS `transform: scale(var(--game-scale))` 等比縮放,JS 監聽 `resize` 重算 scale
  ```js
  const BASE_W = 1100, BASE_H = 618.75;
  function apply() {
    const scale = Math.min(innerWidth / BASE_W, innerHeight / BASE_H);
    document.documentElement.style.setProperty("--game-scale", scale);
  }
  apply();
  window.addEventListener("resize", apply);
  window.addEventListener("orientationchange", apply);
  if (window.visualViewport) window.visualViewport.addEventListener("resize", apply);
  ```
- 所有圖片放在 `./assets/`,可加 `?v=YYYYMMDD-tag` 防快取
- 背景音樂:`背景配樂_燭之武.mp3`(loop, autoplay, 右上音樂切換鈕)
- 字體:`Noto Serif TC`、`Source Han Serif TC`、`Microsoft JhengHei`、`PingFang TC` fallback 序列
- 文字流程:對白用打字機效果(每字 ~18ms)
- 主檔:`index.html` + `style.css` + `script.js` + `背景配樂_燭之武.mp3` + `assets/`

## 三、美術規格

### 3-1 共用風格基底(每張 prompt 都要套這段)

> 電影級水墨重彩繪本插畫,**色彩深邃飽和、光影戲劇強烈、視覺富有層次**(不是淡彩淡墨,要有電影感的明暗對比)。主色調為「**夜色青藍 + 月白**」(冷色系)。暖光**只用在燈火、月光、印章**,單張畫面暖色面積建議不超過 15%。墨線輕盈靈動,人物有質感。
>
> 人物服飾:**春秋時代周禮風**(寬袖深衣、冠冕、麻履、青銅器具)。**不要漢初、唐宋風格**。
>
> 規格:16:9 橫向、高解析度(建議 1792×1024 或 1920×1080)、**不含 Logo、現代物件、火藥火砲**。
>
> 核心氣質:燭之武故事的 8 成都在夜半進行(夜叩寒門 → 夜縋而出 → 夜訪秦營 → 夜半論辯 → 黎明退兵),畫風要把「**整夜未眠的一人外交**」徹底拍出來。

## 四、計分與 Game Over

```js
const START_SCORE = 100;
const OPTION_PENALTY = 25;   // 對白選擇題答錯(扣全域分數)
const SNEAK_PENALTY = 15;    // 潛行失誤
const PERSUADE_WRONG_PENALTY = 15; // 論點卡送出有錯,扣全域分數
```

- **對白選擇題答錯**:扣 25 分。
- **晉文公追擊抉擇(Q6)**:選錯扣 25 分,跳教學頁後**強制回去重選**(retryUntilCorrect)。
- **潛行小遊戲失誤**:扣 15 分,可重試該關。
- **論點卡送出 5 段有錯**:扣 15 分,5 個論述格保留(玩家自己回去修改),**不告訴錯在哪**。
- **論點卡信任度歸 0**:進入 `gameover_persuade.png`(秦伯逐客),點擊重新挑戰。
- **第三場拒絕出使**:直接進入 `gameover.png` Bad Ending(鄭亡),點擊重新選擇。
- **分數扣到 0**:進入 `gameover.png`,可重玩。

## 五、劇情資料結構

```js
const acts = [
  {
    title: "第一場・大軍壓境",
    scene: "scene1_siege.png",
    view: "視角:燭之武",   // 可選,觸發左上「視角」印章
    events: [
      { type: "line", character: "narrator", text: "..." },
      { type: "line", character: "鄭文公", quote: "...", text: "..." },
      { type: "line", character: "narrator", flashback: true, scene: "scene1b_alliance.png", text: "..." },
      { type: "question", label: "...", character: "...", prompt: "...", options: [...], correct: 0, explanation: "..." },
      { type: "branch", label: "...", character: "...", prompt: "...", choices: [{ text, action: "continue"|"badEnding", toast?, unlock? }] },
      { type: "sneak" },     // 觸發潛行小遊戲
      { type: "persuade" }   // 觸發論點卡牌
    ]
  }
];
```

### 事件型別

| `type` | 用途 |
|---|---|
| `line` | 一般對白或旁白 |
| `question` | 單選題,3 個選項 |
| `branch` | 支線分歧,可導向 Bad Ending |
| `sneak` | 觸發夜縋潛行小遊戲 |
| `persuade` | 觸發論點卡牌說服戰 |

### `line` 可加屬性

- `quote`:文言金句,顯示於對話框上層
- `flashback: true`:閃回對白,UI 加 sepia 濾鏡
- `scene: "xxx.png"`:**中途切換背景圖**(尾聲蒙太奇與閃回常用)

### `question` 可加屬性

- `retryUntilCorrect: true`:答錯扣分後強制回到本題,直到答對才前進(Q6 用)
- `wrongExplanations: [string, string, string]`:對應每個選項的「為什麼錯」教學文
- `unlock: "codexId"`:答對解鎖成語圖鑑該則

## 八、論點卡牌玩法詳細規格(本作核心)

### 8-1 設計原則

- **5 段論述,每段 3 張卡選 1**——不是「10 張排序」,而是「依序 5 個 turn 各選一張」
- **每張卡都是「春秋外交常見的合理建議」**——只有一張是燭之武真正會說的,其他兩張也都看起來「不無道理」(賄賂、結盟、訴諸面子、軍情、輿論等)
- **5 段全填滿才一次性驗證**——不是即時對錯,玩家可以隨時回去重選任一段
- **錯誤不告訴哪段錯**——只說「使臣,你說的話寡人聽不進」,信任度 -15,玩家自己回去重新思考
- **正確才揭曉文言原文**——文言變成「**獎勵教材**」,不是進入門檻
- **每段的 3 張卡片 ABC 隨機洗牌**——避免正解永遠是 A、玩家發現規律

### 8-2 信任度量表

- 範圍:0 ~ 100
- 起點:50
- 滿值(100):秦穆公被說服,觸發退兵
- 歸 0:秦伯逐客,進入 `gameover_persuade.png`,可重試
- 視覺:春秋青銅秤桿 + 暗紅指針
