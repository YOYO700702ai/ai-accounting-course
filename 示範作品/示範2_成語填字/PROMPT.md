# 示範 2：成語填字 · Prompt

> 把下面這整段複製貼到 Antigravity 的 Claude / Codex 對話框，
> AI 就會幫你做出一模一樣的遊戲。
> 這個示範**不用 Sheets 題庫**，成語直接寫在程式碼裡，貼上就能用。

---

```
我是國中國文老師，要做一個「成語填字」拖曳遊戲給學生玩，幫我複習常見成語。
請在我目前 IDE 打開的「我的小遊戲」資料夾裡，建一個叫「示範2_成語填字」的子資料夾，做一個 index.html。

═══ 題庫資料來源 ═══
不用外部 CSV，題目直接硬寫在 JS 裡，至少 10 題。
每題格式如下（範例）：

const QUESTIONS = [
  { 成語: '一鳴驚人', 缺字: [1, 2], 答案: ['鳴', '驚'], 意思: '比喻平時默默無聞，突然有驚人表現。' },
  { 成語: '風花雪月', 缺字: [1, 3], 答案: ['花', '月'], 意思: '原指四時景色，後比喻浮華空洞的詩文或男女戀情。' },
  { 成語: '三番五次', 缺字: [1, 3], 答案: ['番', '次'], 意思: '形容次數很多。' },
  { 成語: '一字千金', 缺字: [1, 3], 答案: ['字', '金'], 意思: '形容文字價值極高，或文章極為精妙。' },
  { 成語: '畫龍點睛', 缺字: [1, 3], 答案: ['龍', '睛'], 意思: '比喻在關鍵處用一兩句話點明要旨，使內容生動傳神。' },
  // …請幫我補滿至少 10 題，挑國中常見成語
];

每題的「缺字」是索引（0-3），代表要挖空的位置；答案是按缺字順序對應的字。

═══ 遊戲核心玩法 ═══
1. 一場遊戲共 5 題（從題庫隨機抽，不重複）
2. 每題畫面：
   - 上方大字顯示成語，缺字位置用方框「□」（空格）代替
   - 例：「一 □ 千 □」
   - 下方排 15 張字卡（隨機排列）：包含這題的 2 個正解字 + 13 個干擾字
     （干擾字可以從其他題的字裡抓，或內建一份常用字池）
   - 字卡順序每次都打亂
3. 玩家把字卡用「拖曳」（HTML5 drag-and-drop）拉到上方的空格
4. 判定：
   - 拖對 → 字卡變綠、鎖定在空格內、播一個清脆鈴聲（可用 Web Audio 短音）
   - 拖錯 → 字卡彈回原位、紅光閃 0.2 秒
   - 老師開關：頁面右上角放一個「錯誤是否扣分」的勾選框（預設不扣）
5. 兩個空格都填對 → 中央彈出卡片顯示「✓ 答對！」+ 成語意思 +「下一題」按鈕
6. 5 題全部完成 → 顯示總成績畫面（答對幾題、用了幾次嘗試）+「重新挑戰」按鈕

═══ 視覺風格 ═══
- 古色古香、書卷字帖風
- 背景：米黃宣紙色（#f5ecd7 之類）+ 淡淡的紋理
- 標題與成語：毛筆楷書字型（Google Fonts 用「Noto Serif TC」或「ZCOOL XiaoWei」）
- 空格用紅色方框（像書法字帖的米字格）
- 字卡用淺色木紋或淡黃底 + 黑色楷體
- 答對時空格內字變為紅色（像紅章蓋上去）
- 角落可以放一個小小的紅色印章 emoji 或 SVG 當裝飾

═══ 技術需求 ═══
- 單一 index.html（CSS + JS 全部塞進去，不要拆檔）
- 不依賴任何第三方套件（字型用 Google Fonts CDN 可以）
- 中文 UTF-8 編碼
- 拖曳要同時支援桌機滑鼠 + 平板觸控（touch events）
- 字卡拖到非空格區域要能彈回原位

完成後請用本地瀏覽器幫我打開。
```

## 📝 改造小撇步

裝出來後，可以叫 AI 再改：

- **換科目**：「把成語換成英文片語填空」「改成數學公式填空」「改成歷史朝代填空」
- **加計時**：「每題給 30 秒倒數，時間到算錯」「總時間排行榜，存在 localStorage」
- **改題數**：「改成 10 題一場」「改成可以選簡單(5題)/普通(10題)/挑戰(20題)」
- **加提示**：「加一個『提示』按鈕，按下會先填好一個字但扣 5 分」
- **改視覺**：「改成可愛 Q 版風格，字卡像麻將牌」「改成手機直式版面」「加上翻牌動畫」
- **多人模式**：「改成兩人輪流答題，比誰先過 5 題」
