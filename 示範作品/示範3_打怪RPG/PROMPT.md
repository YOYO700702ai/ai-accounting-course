# 示範 3：打怪 RPG 進階版 · Prompt

> 把下面這整段複製貼到 Antigravity 的 Claude / Codex 對話框，
> AI 就會幫你做出一模一樣的進階版遊戲（含金幣 + 商店 + 道具 + 救命符）。
> 第一次貼之前，**先把 `[請貼上你的 Sheets CSV 網址]` 換成自己的題庫網址**（章節 1 Step 4 拿到的）。

---

```
我是國中老師，要做一個「打怪 RPG 進階版」練習題網頁給學生玩。
請在我目前 IDE 打開的「我的小遊戲」資料夾裡，建一個叫「示範3_打怪RPG」的子資料夾，做一個 index.html。

═══ 題庫資料來源 ═══
請用 fetch 抓這個 Google Sheets 發布的 CSV 網址：

[請貼上你的 Sheets CSV 網址]

CSV 格式：第一列是表頭「題目,A,B,C,D」，A 欄是正解、B/C/D 是錯誤選項
（如果某欄是「空」就忽略那個選項）

═══ 遊戲核心玩法 ═══
1. 玩家固定一個角色（用 🧙 巫師 emoji），HP 100、初始金幣 0
2. 一場遊戲共 5 個關卡，每關 1 隻怪（依序登場）：
   第 1 關 🐺 野狼（HP 100）
   第 2 關 🦂 毒蠍（HP 100）
   第 3 關 👹 惡鬼（HP 100）
   第 4 關 🐲 巨龍（HP 100）
   第 5 關 👑 魔王（HP 100）
3. 戰鬥流程：
   - 螢幕中央顯示「我方角色 vs 怪物」+ 各自 HP 血條
   - 右上角永遠顯示「💰 金幣總額」
   - 怪物頭上跳出一題（從 CSV 隨機抽，選項順序打亂）
   - 下方 4 個選項按鈕（A/B/C/D）
   - 答對 → 角色攻擊動畫，怪物 HP -25 + 跳「-25」傷害數字
           + 玩家獲得 5 金幣（畫面跳「+5 💰」）
   - 答錯 → 怪物攻擊動畫，玩家 HP -25 + 顯示正確答案 2 秒
   - 接著下一題
4. 怪物 HP 歸零 → 短動畫「怪物倒下」+ 額外掉落 20-50 隨機金幣
   → 進入「商店畫面」（最後一關除外）
5. 5 隻怪都打完 → 通關畫面（金色閃光）
   + 顯示「答對幾題 / 剩餘金幣 / 用了幾瓶藥水」+「再玩一次」

═══ 金幣系統 ═══
- 右上角金幣計數器（用 💰 圖示 + 數字）
- 答對 1 題 = +5 金幣
- 擊敗怪物 = 額外 +20 到 +50 隨機金幣（跳大字動畫）

═══ 商店系統（每關之間自動進入）═══
打倒怪物後彈出商店畫面，可買道具（金幣不夠的灰掉）：

| 道具 | 效果 | 價格 |
|------|------|------|
| 🧪 紅藥水 | 補 50 HP（戰鬥中也能用） | 30 金幣 |
| 💎 攻擊水晶 | 下一題答對暴擊：怪物 HP -50 | 40 金幣 |
| 🛡 救命符 | HP 歸零時自動復活，剩 1 滴 HP | 80 金幣 |

- 每樣可重複購買、堆疊
- 商店右下角「進入下一關」按鈕

═══ 死亡與復活機制 ═══
- 玩家 HP 歸零時：
  - 若背包有「🛡 救命符」→ 自動消耗 1 個，HP 回到 1，繼續戰鬥
  - 若沒有 → 自動「重來該關」：怪物 HP 重置、玩家 HP 設成 1（懲罰）
  - 若已經是 HP 1 又死掉且沒救命符 → Game Over 畫面（紅底）+「重新開始」

═══ 背包與隨時補血 ═══
- 戰鬥畫面下方有「🎒 背包」按鈕
- 點開顯示目前持有的道具與數量
- 戰鬥中可以點「🧪 紅藥水」立刻補 50 HP（不消耗回合）
- 「💎 攻擊水晶」會掛在欄位上，下一題答對自動觸發暴擊

═══ 視覺風格 ═══
- 像素風或卡通可愛風，深色背景配明亮重點色
- HP 血條：綠色 → 黃色（< 50%）→ 紅色（< 20% 會閃爍）
- 金幣計數器：金黃色，數字變動時跳一下
- 答對時整個畫面閃綠光 0.2 秒
- 答錯時畫面震動 0.3 秒
- 暴擊時整個畫面閃紅光 + 大字「CRITICAL!」
- 商店畫面：木紋背景 + 道具卡片 + 大字標題「🏪 道具商店」
- 字體可愛清楚，中文用「Noto Sans TC」之類圓潤的字型

═══ 技術需求 ═══
- 單一 index.html（CSS + JS 全部塞進去，不要拆檔）
- 不依賴任何第三方套件，用純 fetch + JavaScript 解析 CSV
- 中文 UTF-8 編碼
- 如果 CSV 抓不到，要有一份內建範例題庫當 fallback（10 題左右），避免空白
- 開啟時自動 fetch 題庫，顯示 loading 動畫
- 金幣、HP、背包用 JavaScript 物件管理，狀態更新時 UI 自動刷新

完成後請用本地瀏覽器幫我打開。
```

## 📝 改造小撇步

裝出來後，可以叫 AI 再改：

- **換主題世界觀**：「把怪物換成 5 個三國武將」「改成寶可夢風格、5 隻精靈」「改成校園鬼故事，5 隻校園怪談」
- **加 Boss 戰機制**：「第 5 關魔王分兩階段，血量打到一半會狂暴 +50% 攻擊」「魔王每 3 題會丟一個必答題，答錯直接 -50 HP」
- **改數值平衡**：「藥水改成 20 金幣 但只補 30 HP」「答對改成 +10 金幣，但答錯扣 30 HP」「怪物 HP 改成 150，玩家攻擊改 30」
- **加更多商店道具**：「加『🔮 透視鏡』50 金，下一題自動排除一個錯誤選項」「加『⚡ 閃電卷軸』60 金，直接秒殺小怪」「加『🍀 幸運草』，下一關金幣 +50%」
- **加成就 / 排行系統**：「加成就『不死流』通關全程沒死過」「結算頁顯示『S/A/B/C』評等（看剩餘 HP 和金幣）」「最後輸入名字，存 localStorage 排行榜」
